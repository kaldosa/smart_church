상품 APIs

상품 목록

상품 목록을 요청한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/goods/?criteria=${criteria}&keyword=${keyword}
criteria : 검색기준 (pay: 유료 [keyword 없음], category: 분류 [BIBLE etc..], name: 상품명 [상품명]))
keyword : 검색어

요청 메소드(Request Method)

GET

요청 변수(Request Parameter)

요청 변수

출력 변수(Response Element)

출력 변수
  • code : 응답코드
  • description : 응답코드에 대한 설명
  • data : 요청 관련 데이터
    • list : 목록
      • id : 상품 ID
      • category : 분류
      • name : 상품명
      • price : 가격 (0 이면 무료)
      • registeredDate : 등록일
      • productKeys : 마켓별 제품 key
        • id : productkey id
        • goods : 상품 id
        • market : 마켓
        • productKey : 제품 key
      • attachment : 리소스
        • id : 리소스 ID
