사용자 APIs

인증코드 발송

회원가입을 위해 사용할 이메일이 사용가능하지 확인을 위한 인증코드를 메일로 발송한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/usersupport/chekMail

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • email : 이메일주소

출력 변수(Response Element)

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명
응답 예제 
{
  "code": 100,
  "reason": "입력 정보가 잘못되었습니다."
}

인증코드 검증

이메일 유효성 검증을 위해 발송한 인증코드를 검증한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/usersupport/verifyCode

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • email : 이메일주소
  • code : 인증코드

출력 변수(Response Element)

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명
응답 예제 
{
  "code": 100,
  "reason": "입력 정보가 잘못되었습니다."
}

가입 신청

사용자 가입 신청을 한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/usersupport/join

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • email : 이메일주소
  • password : 비밀번호
  • name : 이름
  • mobilePhone : 전화번호
  • payed : 기존 유료 앱 사용자유무 (true/false)

출력 변수(Response Element)

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명
응답 예제 
{
  "code": 100,
  "reason": "입력 정보가 잘못되었습니다."
}

임시 비밀번호 요청

비밀번호를 잊었을 때, 임시 비밀번호를 이메일로 발송한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/usersupport/find/password

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • email : 이메일
  • name : 이름

출력 변수(Response Element)

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명
응답 예제 
{
  "code": 204,
  "reason": "입력하신 정보와 일치하는 사용자가 없습니다. \n정확한 정보로 확인 후 다시 입력 부탁드립니다."
}

사용자 uid 요청

사용자의 uid를 요청한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/user/getuid

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • email : email주소

출력 변수(Response Element)

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명
  • data : 요청 관련 데이터
    • uid : 사용자 id
    • email : 사용자 email
    • name : 사용자 이름
    • mobilePhone : 사용자 전화번호
    • enable : 사용자 활성화 유무
    • payed : 유료 앱사용자 유무
    • photo : 사용자 프로필 사진
      • id : 이미지 id
      • realFileName : 파일명
      • parentPath : 경로
      • fileSize : 파일 사이즈

기도노트 업로드

앱 사용자의 기도노트를 서버에 업로드한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/user/{uid}/uploadPrayer
uid : 사용자 id

요청 헤드(Request Header)

Content-Type: application/json;charset=utf-8

요청 메소드(Request Method)

POST

요청 변수(Request Parameter)

요청 변수
  • version : 성경 버전
  • book : 성경책 인덱스
  • chapter : 장 인덱스
  • verse : 절 인덱스
  • subject : 제목
  • content : 내용
  • praise : 기도응답에 대한 소감
  • writtenDate : 작성일 (yyyy.MM.dd 양식)
  • responseDate : 기도 응답 받은 날 (yyyy.MM.dd hh:mm 양식)
  • author : 사용자 id

[
    {

        "chapter":0,

        "praise":"determine",
        "author": 3,
        "writtenDate":"2012.9.27",
        "responseDate":"2012.9.27"
    }
]

출력 변수
  • code : 상태 코드
  • reason : 상태 코드 설명

성경읽기 묵상,기도노트 목록

성경읽기에서 해당 챕터에 대한 묵상,기도노트 목록을 요청한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/user/${uid}/bible?version=${version}&book=${book}&chapter=${chapter}
uid : 사용자 ID
version : 성경 버전 (성경읽기에서만 사용되는 요청 파라미터) book : 성경책 인덱스 (성경읽기에서만 사용되는 요청 파라미터) chapter : 챕터(장) 인덱스 (성경읽기에서만 사용되는 요청 파라미터)

요청 메소드(Request Method)

GET

요청 변수(Request Parameter)

요청 변수

출력 변수(Response Element)

출력 변수
  • code : 응답코드
  • description : 응답코드에 대한 설명
  • data : 요청 관련 데이터
    • plist : 기도노트 목록
      • id : 기도노트 ID
      • version : 성경 버전
      • book : 성경책 인덱스
      • chapter : 장 인덱스
      • verse : 절 인덱스
      • subject : 제목
      • content : 내용
      • praise : 기도응답에 대한 소감
      • writtenDate : 작성일 (yyyy.MM.dd 양식)
      • responseDate : 기도 응답 받은 날 (yyyy.MM.dd 양식)
      • author : 사용자 id
    • mlist : 묵상노트 목록
      • id : 묵상노트 ID
      • version : 성경 버전
      • book : 성경책 인덱스
      • chapter : 장 인덱스
      • startVerse : 절 시작 인덱스
      • endVerse : 절 끝 인덱스
      • subject : 제목
      • verses : 성경구절
      • content : 내용
      • determine : 결심한 내용
      • writtenDate : 작성일 (yyyy.MM.dd hh:mm 양식)
      • author : 사용자 id

묵상노트 목록

사용자의 묵상노트 목록을 요청한다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/user/${uid}/meditate?page=${page}
uid : 사용자 ID
page : 페이지 (default : 1)

요청 메소드(Request Method)

GET

요청 변수(Request Parameter)

요청 변수

출력 변수(Response Element)

출력 변수
  • code : 응답코드
  • description : 응답코드에 대한 설명
  • data : 요청 관련 데이터
    • list : 묵상노트 목록
      • id : 묵상노트 ID
      • version : 성경 버전
      • book : 성경책 인덱스
      • chapter : 장 인덱스
      • startVerse : 절 시작 인덱스
      • endVerse : 절 끝 인덱스
      • subject : 제목
      • verses : 성경구절
      • content : 내용
      • determine : 결심한 내용
      • writtenDate : 작성일 (yyyy.MM.dd hh:mm 양식)
      • author : 사용자 id
    • paginate : 페이징 정보
      • totalItem : 전체 아템 수
      • totalPage : 전체 페이지 수
      • itemPerPage : 페이지당 아이템 출력 개수
      • page : 현재 페이지
