교회 검색

사용자의 교회 설정을 위한 교회 검색 API

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/church/?page=${page}&criteria=name&keyword=${keyword}
page : 페이지 (default : 1)
criteria : 검색기준 (name : 교회이름 , pastor : 담임목사, address : 교회주소)
keyword : 검색어

요청 메소드(Request Method)

GET

요청 변수(Request Parameter)

요청 변수

출력 변수(Response Element)

출력 변수
  • code : 응답코드
  • description : 응답코드에 대한 설명
  • data : 요청 관련 데이터
    • list : 교회 목록
      • id : 교회 ID
      • name : 교회 이름
      • pastor : 담임 목사
      • address : 주소
      • contact : 전화번호
      • serviceId : 우리교회 서비스 Id (ID 값이 0 이면 우리교회 서비스중이 아닌 교회)
      • serviceStatus (option) : 우리교회 서비스 상태 (WAITING(서비스 승인 대기), SERVICE(서비스중), STOP(서비스 중지))
    • paginate : 페이징 정보
      • totalItem : 전체 아템 수
      • totalPage : 전체 페이지 수
      • itemPerPage : 페이지당 아이템 출력 개수
      • page : 현재 페이지

우리교회 서비스 검색

우리교회 서비스중인 교회 목록을 가져온다.

요청 주소(Request URL)

https://www.smartchurch.co.kr/apis/church/service?page=${page}&criteria=name&keyword=${keyword}
page : 페이지 (default : 1)
criteria : 검색기준 (이름)
keyword : 검색어 (교회명)

요청 메소드(Request Method)

GET

요청 변수(Request Parameter)

요청 변수

출력 변수(Response Element)

출력 변수
  • code : 응답코드
  • description : 응답코드에 대한 설명
  • data : 요청 관련 데이터
    • list : 교회 목록
      • id : 우리교회 서비스 ID
      • path : 우리교회 root path
      • status : 우리교회 서비스 상태 (SERVICE, STOP)
      • createdDate : 우리교회 서비스 신청일
      • modifiedDate : 우리교회 서비스 승인날짜
      • enabled (현재는 사용안함) : true
      • churchMeta : 교회 기본 정보
        • church : 교회 ID
        • name : 교회 이름
        • pastor : 담임 목사
        • contact : 전화번호
        • address : 교회주소
      • applicant : 우리교회 서비스 신청인 (관리자)
        • uid : user id
        • email: email
        • name: user name
      • logo : 로고 이미지
        • id : 이미지 id
        • realFileName : 파일명
        • parentPath : 경로
        • fileSize : 파일 사이즈
    • paginate : 페이징 정보
      • totalItem : 전체 아템 수
      • totalPage : 전체 페이지 수
      • itemPerPage : 페이지당 아이템 출력 개수
      • page : 현재 페이지
